# NodeMCU-ESP8266+Nokia5110+DHT11+DS18B20
My first project Arduino

## Components
* NodeMCU 1.0 - ESP8266 12-E
* LED Screen Nokia 5110
* DHT11
* DS18B20
* Switch buttom
* Two resistors of 10K in parallels mode
* 20 Jumper Wire
* Two Mini breadboard
